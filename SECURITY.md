# Security Policy

## Reporting a Vulnerability
- Please email security reports to `security@example.com`.
- Include a description, reproduction steps, and potential impact.
- Do not open public issues for security problems.

## Supported Versions
- `main` branch receives security fixes.

## Expectations
- We aim to acknowledge reports within 72 hours.
- Disclosures are coordinated; please do not publish details until a fix is available.
